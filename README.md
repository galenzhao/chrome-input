## 插件介绍：Input Improve Helper

**Input Improve Helper** 是一个为网页输入框提效的 Chrome 插件。  
在任意网站的文本输入区（输入框、文本域、部分编辑器）旁边，自动浮现一个「一键优化」按钮。你可以在插件配置页里为它接入任意 OpenAI 兼容接口，并配置多种“优化方式”（不同模型 + Prompt），比如「润色成更礼貌的中文、压缩为要点、翻译成英文」等。  
写完一句话点一下，插件会调用你配置的模型优化这段文本，并直接替换回原来的输入框，减少来回复制粘贴的时间。

### 近期更新

- **支持 Token 统计**：同时提供**单次调用** token 用量（如 prompt / completion / total）与**历史累计总数**统计，方便你持续评估成本与效果。
- **支持 CodeMirror 文本框**：在使用 CodeMirror 的网站/应用中也能正常识别编辑区并进行一键优化。

### 主要特性

- **全站适配**：支持大部分网页的 `input` / `textarea` / `contenteditable` 文本输入区域（默认排除密码框）。
- **多种优化方式**：在配置页自由添加多种优化模板，每种模板可指定模型和 Prompt，支持不同语气/语言/风格。
- **OpenAI 兼容接口**：支持 `POST /v1/chat/completions` 风格的兼容接口，既可以接官方 OpenAI，也可以接自建代理。
- **自动拉取模型列表**：优先从 `/v1/models` 获取模型列表，用下拉框选择；不兼容时自动降级为手动填写模型名。
- **按钮位置可配置**：支持跟随输入框四个角、或固定在页面四个角，避免与其他插件 UI 冲突。
- **隐私友好**：API Key 仅保存在本地浏览器扩展存储中，只用于发起你自己的接口请求，不上传到第三方服务器。

### 隐私政策（Chrome Web Store 必填）

Chrome Web Store 审核要求在「Privacy」页的**指定字段**填写一个可公开访问的隐私政策链接（仅写在描述里不算）。  
仓库内已提供隐私政策页面：`docs/privacy-policy.html`（以及文本版 `PRIVACY_POLICY.md`）。

### 使用方法（简要）

1. 在 Options 页面中配置：
   - Base URL：你的 OpenAI 兼容接口根地址（如 `https://api.openai.com` 或自建代理）。
   - API Key：对应服务的密钥。
   - 可点击「授权当前域名」「刷新模型列表」「验证接口」确保配置可用。
2. 在「优化方式」区域为不同场景添加多种模板（名称 + 模型 + Prompt）。
3. 在「按钮位置」区域选择按钮跟随输入框或固定在页面角落，避免与其它插件 UI 冲突。
4. 回到任意网页，在文本框内输入内容，点击旁边的「优」按钮，选择优化方式，插件会调用接口并用返回结果替换原始文本。
