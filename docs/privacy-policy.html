Privacy Policy — Input Improve Helper

Effective date: 2026-03-27

Summary

  • No developer-run servers: this extension does not operate a backend service.
  • No selling/sharing for ads: we do not sell or share user data for advertising.
  • API calls: text you choose to optimize is sent to the API endpoint you configure.
  • Local storage: your API configuration and settings are stored locally in Chrome extension storage.

What data is processed

  • User-provided text content: when you click optimize, the extension reads the current text from the focused input/textarea/contenteditable/editor and sends it to your configured API endpoint to generate an improved version.
  • API configuration: base URL, API key, preferred model, prompts/modes.
  • Usage statistics: token usage totals and last-call usage (if returned by your API provider), stored locally.

How data is used

  • Provide functionality: send your selected text to the configured API and return improved text.
  • Persist settings: store configuration and UI preferences in chrome.storage.local.

Data sharing / third parties

The extension transmits your text to the third-party API provider you configure (for example, OpenAI or an OpenAI-compatible provider you select). Their handling of data is governed by their privacy policy and terms.

We do not otherwise disclose data to third parties, and we do not use data for advertising.

Storage and retention

  • On-device: configuration, prompts/modes, UI settings, and usage stats are stored locally until you remove them (for example, by uninstalling the extension or clearing extension data).
  • API provider retention: depends on the provider you configured.

Security

Your API key is stored in Chrome extension storage on your device. Treat it as sensitive, and consider using a restricted key if your provider supports it.

Children’s privacy

This extension is not directed to children under 13, and we do not knowingly collect personal information from children.

Changes

We may update this policy from time to time by posting a new version on this page with a revised effective date.
